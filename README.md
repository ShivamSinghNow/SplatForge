# SplatForge

> 🚧 Early-stage project — scaffolding in progress.

SplatForge is a work-in-progress project that uses [ChromaDB](https://www.trychroma.com/)
as a local vector store for embeddings / semantic search.

## Status

The repository is in its initial setup phase. Source code will be added here.

## Local data

The vector store lives in `chroma_db/` and is **not** committed to version control
(it's generated data). It will be created/populated locally when you run the project.

## Getting started

```bash
# clone
git clone https://github.com/ShivamSinghNow/SplatForge.git
cd SplatForge

# (once a Python environment is set up)
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt   # to be added
```
